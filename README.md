# Italian Learning Application for Urdu Speakers

Welcome to the Italian Learning Application, specifically designed for Urdu speakers who want to learn Italian efficiently.

## Features
- **User-Friendly Interface**: Easy navigation and interactive lessons for effective learning.
- **Vocabulary Builder**: Flashcards and exercises to expand your Italian vocabulary.
- **Grammar Guide**: Comprehensive guide to Italian grammar with examples and practice exercises.
- **Audio Pronunciation**: Native speaker pronunciations to help improve your spoken Italian.
- **Quizzes and Assessments**: Test your knowledge with quizzes at the end of each lesson.

## Installation
To use this application, follow these steps:
1. Clone the repository: `git clone https://github.com/FA-05/urduo2.git`
2. Navigate to the project directory: `cd urduo2`
3. Install the necessary packages: `npm install`
4. Start the application: `npm start`

## Usage
- Select a lesson from the main menu.
- Complete exercises and quizzes to track your progress.
- Use the vocabulary builder to learn new words daily.
